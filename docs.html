Add Alertify to your project

To use Alertify just add the follwing lines to your code.
<!--Stylesheet-->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/vikkycreations/alertify/dist/alertify.min.css" /> 
 
<!--core js file-->
<script src="https://cdn.jsdelivr.net/gh/vikkycreations/alertify/dist/alertify.min.js"></script>

Initialisation & usage

To initialize an alert you have to create an new instance of the Alertify object.
var customerAlert = new alertify("Hello World!.."); 
see just as simple as that.

Customization

Alertify has various properties to customize the popup based on your interest. To customize the popup you need to pass in a JSON object with the options and their values. The various options a title, text, type, position, showSuccessbtn, showCancelbtn, openBy, closeBy.
  • title
    name: title
    type: string
    defaultValue: ''

    //code
    var customerAlert = new alertify({  
      title: "Welcome",
      ...
    });
    Give your popup a title based on purpose of it.
  • text
    name: text
    type: string
    defaultValue: ''

    //code
    var customerAlert = new alertify({ 
      text: "Hello World!..",
      ...
    });
    Message shown on the popup to the users.
  • type
    name: type
    type: string
    defaultValue: ''
    values: 'info', 'success', 'warning', 'danger' 

    //code
    var customerAlert = new alertify({
      type: "success",
      ...
    });
    Make popups based on the nature of the message.
         
  • position
    name: position
    type: string
    defaultValue: 'top'
    values: 'top', 'center', 'bottom' 

    //code
    var customerAlert = new alertify({
      position: 'center',
      ...
    });
    Align your popup vertically.
       
  • showSuccessbtn
    name: showSuccessbtn
    type: Boolean
    defaultValue: true
    values: true, false

    //code
    var customerAlert = new alertify({ 
      showSuccessbtn: false,
      ...
    });
    Toggle the display of success button.
     
  • showCancelbtn
    name: showCancelbtn
    type: Boolean
    defaultValue: false
    values: true, false

    //code
    var customerAlert = new alertify({ 
      showCancelbtn: true,
      ...
    });
    Toggle the display of cancel button.
     
  • openBy
    name: openBy
    type: string
    defaultValue: 'ZoomIn'
    values: 'ZoomIn', 'SlideUpIn', 'SlideDownIn', 'SlideLeftIn', 'SlideRightIn' 

    //code
    var customerAlert = new alertify({ 
      openBy: 'SlideUpIn',
      ...
    });
    Make your popup appear in a cool way!
         
  • closeBy
    name: closeBy
    type: string
    defaultValue: 'ZoomOut'
    values: 'ZoomOut', 'SlideUpOut', 'SlideDownOut', 'SlideLeftOut', 'SlideRightOut' 

    //code
    var customerAlert = new alertify({
      closeBy: 'SlideUpOut',
      ...
    });
    Make your popup disappear in a cool way!
